# Ravenous

React app to search for restaurants using the Yelp API.

## Description

This is project from CodeAcademy's "Create a Front-End App with React" Path

## Getting started

### Executing program

```
git clone https://github.com/nesherson/ravenous.git
cd ravenous
npm install

```
